## About

I'm a senior studying **Computer Engineering at NC State University**, with a focus on embedded firmware, digital hardware, robotics, and power electronics.

I enjoy understanding systems from the lowest level up—from electrical schematics and hardware interfaces to firmware architecture and system behavior.

My work usually involves some combination of:

* writing firmware in C and C++
* designing state machines and control logic
* configuring microcontroller peripherals
* debugging hardware and software together
* developing FPGA designs in Verilog
* building systems that respond to real-world inputs

### Digital Hardware

Designing finite-state machines, datapaths, arithmetic units, and control modules using Verilog, simulation, testbenches, and FPGA workflows.

</td>

<tr>

### Power Electronics

</td>
<td width="50%" valign="top">
